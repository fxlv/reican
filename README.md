# Loggy
Do funny stuff with logs.
Not much here yet.

## Timestamps
Supported formats:
* 2015-10-30T20:20:11.563278+00:00 
* 2015-10-31 11:13:43.541912
* 1446236678.247

## Virtualenv
For the first time:
```
sudo pip install virtualenv
virtualenv venv
```
Then Every time:
```
source venv/bin/activate
```

## Dependencies
Inside the virtualenv you might want to have:
* pytest
* backports.lzma 
* monkeypatch
